# Chrome Web Store Listing

## Short Description (132 characters max)
A beautiful new tab page with accordion-style spaces to organize your bookmarks. Import from Arc Browser. Syncs with Chrome bookmarks.

## Full Description

Transform your new tab into a powerful bookmark organizer inspired by Arc Browser's spaces.

**Features:**

• **Accordion Layout** — Organize bookmarks into collapsible spaces that expand with a click
• **Arc Browser Import** — Import your spaces and bookmarks directly from Arc's StorableSidebar.json file, preserving colors and folder structure
• **Chrome Sync** — Two-way sync with Chrome's native bookmarks bar. Changes in either place stay in sync
• **Color-Coded Spaces** — Each space has its own accent color for easy visual organization
• **Drag & Drop** — Move bookmarks and folders between spaces effortlessly
• **Folder Organization** — Create nested folders with expand/collapse all controls
• **Standard Import** — Also supports importing standard HTML bookmark files from any browser

**Getting Started:**

1. Install the extension
2. Click the Import button to bring in your bookmarks
3. For Arc users: Import your StorableSidebar.json file (found in ~/Library/Application Support/Arc/)
4. Organize your bookmarks into spaces
5. Your bookmarks automatically sync to Chrome's Bookmarks Bar

**Privacy First:**

All your data stays on your device. No accounts, no tracking, no external servers. Your bookmarks are yours.
